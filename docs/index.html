◆作品◆

《Zu-Buono!》

"ずぼーの" = ずぼらでBuono(イタリア語でおいしい)なレシピを投稿 & 閲覧するwebアプリケーションです。

一般のレシピ投稿サイトと違うのは、投稿できる情報が極めて制限されていることです。
例えば材料は5種類まで、作り方は200文字までしか投稿できません。
これにより、投稿されるレシピは作るのに手間がかからず、作りやすいレシピに限られます。

リンク：Zu-Buono!

作品説明

  • テーマ

    • ずぼらなレシピを気軽に投稿

  • コンセプト

    • 「手間をかけるだけが料理じゃない」

  • 機能
    • レシピ投稿機能

      • レシピの内容をwebページのフォームに記入して、投稿できます。

      • 料理画像投稿機能 (実装予定)

        • レシピを投稿するとき、料理の画像も投稿できます

    • レシピ閲覧機能

      • 投稿されたレシピの一覧からレシピを選んで情報を閲覧できます。

  • 使用技術
    • Python 3.11.0
    • HTML
    • JavaScript
    • AWS Amplify
    • AWS Lambda
    • AWS DynamoDB
    • AWS Amazon API Gateway

  • 制作期間

    • 1週間

  • 動機

    • Twitterで"ずぼらレシピ"のツイートが人気なのを見て、「気軽なレシピを集めたサイトがあれば面白いかも」と思ったのがきっかけです。

  • アピールポイント
    • 一般のレシピ投稿サイトと違い、ずぼらなレシピしか投稿できないので、気軽に投稿できる
    • AWS Amplify, AWS Lambda, AWS Amazon API Gateway等のサービスを活用することで、開発スピードを早めた

  • 苦労したポイント
    • NoSQL DB (DynamoDB)の扱い方に苦労した
    • 異なるサービス間の情報のやり取り(APIリクエスト・レスポンス、JSONオブジェクトの操作)に苦労した
    • 事前にJavaScriptの知識が全くない状態で制作を開始したが、構想段階で想像していたよりもJavaScriptを書く量が多く苦労した


《saitama_neko》

埼玉県の公式HPで公開している譲渡用猫情報を定期的にツイートするTwitter Botを動かすプログラムです。

実際の稼働に関しては、埼玉県のHPにおける"著作権、リンク等の規約"を満たさないため控えておりますが、私的利用の範囲(鍵付きアカウント)にて稼働テスト済みです。

リンク： GitHubページ

作品説明

  • テーマ

    • 譲渡用猫のための飼い主探し

  • コンセプト

    • 「迷える猫にいち早く救いの手を」

  • 機能

    • 毎日午前09:00に埼玉県のHPに掲載されている譲渡用猫の情報をまとめて自動的にツイートします

    ツイート例：
    【2023/xx/xx】
    今日の埼玉県北部・西部地区の譲渡用猫情報
    募集中 : x 匹
    お見合い中 : x 匹
    飼い主さん決定 : x 匹

  • 使用技術
    • Python 3.11.0
    • git version 2.39.0.windows.1
    • AWS (EC2)

  • 制作期間

    • 3週間

  • 動機

    • この作品を制作する時期に、実家（長野県）に野良猫が迷い込んできた。その際、野良猫と自治体との関わりについて調べ、興味を持ちました。
    その後、自分が住んでいる埼玉県でも譲渡用猫の飼い主を探している事を知り、それをTwitterで広く知らせることでいち早く飼い主を見つける事ができると考えました。

  • アピールポイント
    • Beautiful Soupによるwebスクレイピングを定期実行することでwebサイトの情報(譲渡用猫情報)の監視を実現
    • TwitterAPIを利用することでTwitterアカウントの自動操作、定期ツイートを実現
    • 情報を公開する場所をTwitterとしたことで、多くの人に気軽に閲覧してもらえるようにした

  • 苦労したポイント
    • 持っていた参考書のみではBeautiful Soupの使い方がわからず、有志が翻訳したリファレンス等を調べて自分で理解する必要があった
    • 同じ内容を連続でツイートできないというTwitterの仕様でエラーが発生。簡単な理由ながら原因を特定するのに苦労した
