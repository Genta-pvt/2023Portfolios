◆作品◆

《saitama_neko》

埼玉県の公式HPで公開している譲渡用猫情報を定期的にツイートするTwitter Botを動かすプログラムです。

実際の稼働に関しては著作権法上の問題と埼玉県のHPにおける著作権、リンク等の規約に反するため控えておりますが、私的利用の範囲(鍵付きアカウント)にて稼働テスト済みです。

※ここにリンクを貼る※

作品説明

  • テーマ

    • 譲渡用猫のための飼い主探し

  • コンセプト

    • 「迷える猫にいち早く救いの手を」

  • 機能

    • 毎日午前09:00に埼玉県のHPに掲載されている譲渡用猫の情報をまとめて自動的にツイートします

    ツイート例：
    【2023/xx/xx】
    今日の埼玉県北部・西部地区の譲渡用猫情報
    募集中 : x 匹
    お見合い中 : x 匹
    飼い主さん決定 : x 匹

  • 使用技術
    • Python 3.11.0
    • git version 2.39.0.windows.1
    • AWS (EC2)
  • 制作期間

    • 3週間

  • 動機

    • この作品を制作する時期に、実家（長野県）に野良猫が迷い込んできた。野良猫と自治体との関わりについて調べたところ、興味を持ち制作に及ぶ。

    その後、自分が住んでいる埼玉県でも譲渡用猫の飼い主を探している事を知り、それをTwitterで広く知らせることでいち早く飼い主を見つける事ができると考えた。

  • アピールポイント
    • Beautiful Soupによるwebスクレイピングを定期実行することでwebサイトの情報(譲渡用猫情報)の監視を実現
    • TwitterAPIを利用することでTwitterアカウントの自動操作、定期ツイートを実現
    • 情報を公開する場所をTwitterとしたことで、多くの人に気軽に閲覧してもらえるようにした
  • 苦労したポイント
    • 持っていた参考書のみではBeautiful Soupの使い方がわからず、有志が翻訳したリファレンス等を調べて自分で理解する必要があった
    • 同じ内容を連続でツイートできないというTwitterの仕様でエラーが発生。簡単な理由ながら原因を特定するのに苦労した
